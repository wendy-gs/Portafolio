Pokemón Go

Data Lovers

Project information

  • Category: Web design
  • Client: Laboratoria
  • Project date: Setiembre, 2020
  • Project URL: https://wendy-gs.github.io/LIM013-data-lovers/src/

Es una aplicación web donde el usuario podrá ver la información de 251 pokemón de la región Kanto y Johto donde además podrá filtrar y ordenar los poderes de los Pokémon.
Tecnologías usadas: HTML, CSS, Vanilla JavaScript, testing con Jest y Figma
